# Generated Documentation

This section contains generated documentation from source code.

## Structure

Each project generates a single Markdown reference file:

- PhotoArchive.Api.md
- PhotoArchive.Core.md
- PhotoArchive.Data.md
- PhotoArchive.Tools.md

## Source

These files are generated from XML documentation comments using xml2doc.

## Purpose

Provide reference documentation derived directly from the codebase.

## Regeneration

Run:

```
./scripts/build-docs.ps1
```

to rebuild all generated documentation.
